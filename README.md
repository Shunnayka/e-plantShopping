# e-plantShopping

## Paradise Nursery - Tienda de Plantas en Línea

Este es el proyecto final para el curso de desarrollo de aplicaciones de IBM. La aplicación es una tienda virtual de plantas de interior que permite a los usuarios explorar una variedad de plantas organizadas por categorías, agregarlas a un carrito de compras, actualizar sus cantidades de forma dinámica y visualizar el costo total en tiempo real.

### Características Principales:
* **Página de Inicio:** Bienvenida atractiva con un botón de inicio para explorar el catálogo.
* **Barra de Navegación:** Acceso rápido a las secciones y un contador dinámico que muestra los artículos en el carrito.
* **Catálogo de Productos:** Tarjetas detalladas por categoría con imágenes, descripciones, precios y botones de acción.
* **Carrito de Compras:** Gestión completa de artículos (incrementar, decrementar, eliminar) con cálculo automático de subtotales y totales.

### Tecnologías Utilizadas:
* React (Componentes funcionales y Hooks)
* Redux Toolkit (Gestión de estado global)
* Vite (Entorno de desarrollo rápido)
* CSS3 (Diseño responsivo y estilizado)
